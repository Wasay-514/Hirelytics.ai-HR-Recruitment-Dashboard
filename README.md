# Hirelytics

Interactive HR analytics dashboard built using Power BI and PostgreSQL to analyze employee data, workforce trends, salary insights, and departmental performance through dynamic visualizations and DAX measures.

---

## 📌 Project Workflow
1. Collected HR employee dataset  
2. Created PostgreSQL database and tables  
3. Imported CSV data into PostgreSQL  
4. Connected PostgreSQL with Power BI using DirectQuery  
5. Performed data cleaning and transformation in Power Query  
6. Created DAX measures and KPIs  
7. Designed interactive HR analytics dashboard  

---

## 🛠️ Technologies Used
- Power BI  
- PostgreSQL  
- SQL  
- Power Query  
- DAX  

---

## 📊 Dashboard Features
- Employee distribution analysis  
- Department-wise insights  
- Salary and workforce trends  
- Gender analysis  
- KPI cards and interactive visuals  
- Dynamic filtering and slicers  

---

## 📈 Key Insights

- Department performance overview
